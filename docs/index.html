plan-viewer
GitHub
Claude Code Plugin

技術の計画を、
誰でも読めるかたちに

plan mode で生成した設計計画を、非エンジニアにも分かる HTML ページに自動変換。スマホでいつでも確認できる。

$ claude --plugin-dir ~/plan-viewer-plugin
Features

シンプルに、確実に

⚡

自動ゲートフック

ExitPlanMode の実行前に自動チェック。HTML を生成し忘れることがない。

🎨

Zenn ダークテーマ

catnose氏が設計した、読みやすく美しいダークテーマをベースに構築。

💬

アノテーション機能

スマホ上でテキストを選択して質問を送信。読み手のフィードバックを収集できる。

📦

ゼロ依存

外部ライブラリ不要。Python 標準ライブラリだけで HTTP サーバーが動く。

How it works

使い方

プラグインを読み込む

claude --plugin-dir ~/plan-viewer-plugin で起動するだけ。

plan mode で計画を立てる

普段通りに plan mode を使う。ExitPlanMode の直前にフックが発火し、HTML 生成を促す。

スマホで確認

HTTP サーバーを起動して、tailscale 等のネットワーク経由でスマホからアクセス。

生成された HTML をスマホで表示したイメージ
Install

セットアップ

# リポジトリをクローン
$ git clone https://github.com/aiharataketo/plan-viewer-plugin

# プラグインを指定して起動
$ claude --plugin-dir ~/plan-viewer-plugin

# 初回セットアップ（Claude Code 内で実行）
❯ /plan-viewer:setup

# サーバー起動
❯ /plan-viewer:serve
